# client-server-cli

Express server for client side.

## Getting Started

*  Install with npm: `npm install @foxythemes/client-server-cli -D`
*  Run it with `clientserver`

## Docs

```	

clientserver --src ./src --not-found pages-404.html --port 8080

Basic options

-s, --src            Sources path
-n, --not-found      Not found page file
-p, --port           Port


```
